# AndroidCalculatorApp
【勉強会資料】iPhoneの電卓アプリをAndroidで作ってみよう.

## build
libgdxのプロジェクトを実行するコマンド一覧
~/brain-trainingをカレントディレクトリとして

.apkとして実行する場合：
gradlew android:installDebug android:run

.exeとして実行する場合：
gradlew desktop:run
